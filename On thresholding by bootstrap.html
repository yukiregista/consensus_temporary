On thresholding by bootstrap

Setting

The situation depicted in the following shows how we compute estimates and bootstrap support.

For simplicity of notation, we only consider topological information and identify tree θθ with its set of (internal) bipartitions ({b1,…,bn−2}{b1,…,bn−2}).

In order to make theoretical arguments easy, we consider a bit more idealized situation, where we obtain infinite number of bootstrap trees. Then the situation can be depicted as follows:

In this situation, XX is the only random element.

We are interested in the relationship Pr(Yi=t∣Si=s)Pr(Yi=t∣Si=s) or Pr(Yik=1∣Sik=sik)Pr(Yik=1∣Sik=sik). In other words, the probability of certain clade(s) to be true given only the value of bootstrap support(s). And how this information can improve the original estimator ^θ(Xi)θ^(Xi).
