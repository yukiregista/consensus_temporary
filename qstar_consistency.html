Q, Q mode consistency

This notebook is a trial to confirm theoretical results for Q* consensus and Q* mode consensus.

  • Q* mode consensus: When constructing the set $Q_{maj}$, instead of counting number of occurences, we do mode estimation and store the topology of the mode.

The dataset is produced in the following manner:

  • Species tree is randomly generated using birth-death process.
  • Gene trees are obtained by Multispecies Coalescent model with population size $1$ for every edge. (Corresponds to properly scaled version of gene trees?)
    • number of gene trees: one of [50, 100, 300, 500]
    • number of samples for each setting: 50 samples.
    • the total number of gene trees produced is thus 50 * (50 + 100 + 300 + 500) = 47500
  • Consensus either by Q* or Q* mode is constructed using gene trees.

ResultDF = pd.DataFrame(index = [i for i in range(20)])
for sample_size in tqdm(sample_sizes):
    with open(f"simulation_data/MSC_1015/qstarmode_{sample_size}_dict.pkl", "rb") as f:
        qstarmode_dict = pickle.load(f)
    with open(f"simulation_data/MSC_1015/qstar_{sample_size}_dict.pkl", "rb") as f:
        qstar_dict = pickle.load(f)
    
    sds_dict_qstarmode = dict()
    sds_dict_qstar = dict()
    for i in trange(20):
        true = Tree_with_support.get(data=true_trees[i].as_string("newick", suppress_rooting=True), schema="newick", taxon_namespace = true_trees[i].taxon_namespace)
        sds_qstar = []; sds_qstarmode = [];
        for j in range(50):
            qstarmode = Tree_with_support(qstarmode_dict[(i, sample_size, j)], taxon_namespace = true.taxon_namespace)
            #sds.append(distance_p4(true, qstarmode))
            sds_qstarmode.append(dendropy.calculate.treecompare.symmetric_difference(true, qstarmode))
            qstar = Tree_with_support(qstar_dict[(i, sample_size, j)], taxon_namespace = true.taxon_namespace)
            #sds.append(distance_p4(true, qstarmode))
            sds_qstar.append(dendropy.calculate.treecompare.symmetric_difference(true, qstar))
        sds_dict_qstarmode[i] = sds_qstarmode
        sds_dict_qstar[i] = sds_qstar
    sds_qstarmode_mean = dict(); sds_qstarmode_std = dict()
    for key, val in sds_dict_qstarmode.items():
        sds_qstarmode_mean[key] = np.mean(val)
        sds_qstarmode_std[key] = np.std(val)
    sds_qstar_mean = dict(); sds_qstar_std = dict()
    for key, val in sds_dict_qstar.items():
        sds_qstar_mean[key] = np.mean(val)
        sds_qstar_std[key] = np.std(val)
    ResultDF[f"qstar_mean_{sample_size}"] = sds_qstar_mean
    ResultDF[f"qstar_std_{sample_size}"] = sds_qstar_std
    ResultDF[f"qstarmode_mean_{sample_size}"] = sds_qstarmode_mean
    ResultDF[f"qstarmode_std_{sample_size}"] = sds_qstarmode_std
